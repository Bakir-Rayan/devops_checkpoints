OS used ==> Ubuntu 20.04
java version used ==> openjdk 11.0.11

I- So to install jenkins, first we need check if java is installed the we proced to installinf jenkins and anabele it 

    1- Check if java is installed :
    
        java --version
    
    if the ouput is like below : 
    
        openjdk 11.0.11 2021-04-20
        OpenJDK Runtime Environment (build 11.0.11+9-Ubuntu-0ubuntu2.20.04)
        OpenJDK 64-Bit Server VM (build 11.0.11+9-Ubuntu-0ubuntu2.20.04, mixed mode, sharing)
    
    that mean java is installed, otherwise we can install it by :

        sudo apt update     # we update the repositories
        sudo apt install openjdk-11-jdk     # we install java (the recommended version for the last release of Jenkins)
        java -version       # we check the installation of java

    2- Installing jenkins :

        wget -q -O - https://pkg.jenkins.io/debian-stable/jenkins.io.key | sudo apt-key add -       # we add the GPG key of the jenkins repositorie 
                                                                                                    # ( we should get an OK response if everything worked correctly)
        sudo sh -c 'echo deb http://pkg.jenkins.io/debian-stable binary/ > /etc/apt/sources.list.d/jenkins.list'        
                                    # we add the repositorie link to the sources list directory of ubuntu so we can update jenkins within other updates automaticly
        sudo apt-get update     # we update again ubuntu so we can get the new list of new realese including the last realese of jenkins 
        sudo apt-get install jenkins        # we intall jenkins
